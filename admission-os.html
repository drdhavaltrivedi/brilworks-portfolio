Project Overview

Admission OS is a multi-agent platform designed for Open Admits to automate the intensive $10,000+ MBA consulting process. Instead of a simple chatbot, it orchestrates specialized AI agents around a central intelligence layer — the Narrative Anchor Object (NAO).

The Market Gap

The elite admissions consulting market is gated by high human costs ($5K–$15K). Existing AI tools are generic "essay graders" that lack strategic positioning, cross-document memory, and logic validation.

The Brilworks Solution

We engineered a "Consulting Team in a Box" using Google Gemini. The system analyzes, position, and tracks an applicant's story across 8 document modules, ensuring 100% consistency and strategic alignment.

Shared Intelligence

Narrative Anchor Object (NAO)

The core innovation is a machine-readable JSON object that serves as the platform's "shared memory." It prevents AI hallucination and ensures every agent knows exactly what the others are doing.

narrative_anchor_v2.json
{
  "positioning": "Global FinTech Leader",
  "core_narrative": "Bridging trade gaps in emerging markets",
  "differentiators": ["Operational Resilience", "Cultural IQ"],
  "goal_bridge": {
    "current": "Product Lead at Stripe",
    "mba_need": "Strategic logic in macro-economics"
  },
  "voice_fingerprint": { "tone": "Pragmatic", "complexity": 0.74 }
}

The 15s Workflow

When a student clicks "Evaluate," our proprietary orchestration DAG (Directed Acyclic Graph) executes branches in parallel to minimize latency.

T+0ms
Context Loading
Fetch student profile and agent memory from Supabase edge stores.
T+1.5s
Segmentation
Gemini Flash parses the document into thematic blocks (Action vs Impact).
T+4.0s
Parallel Analysis
Scoring Engine and Story Architect fire simultaneously using cached prompt prefixes.
T+7.0s
Strategic Synthesis
Strategic Advisor merges all results into a unified priority list.
T+15s
Delivery
Results pushed via real-time subscription. Persisted for cross-doc analysis.
